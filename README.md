# Alta: Mindset, Refined
Alta is a high-performance mental-wellness platform designed for those who value aesthetic minimalism and cognitive science. Built with a "Midnight Command" design language, Alta leverages edge-computing AI and biometric integrations to deliver personalized psychological reframing in real-time.

# 🌌 The Aesthetic
Theme: #4b30cf Indigo Accent.

Palette: Deep Onyx (#000000) backgrounds with softer borders and consistent token-based success/error states.

Visuals: Animated breathing circles, distortion badges, and high-fidelity video vision boards.

# 🚀 Key Features
1. AI CBT Coach & Cognitive Reframing
Conversational Interface: Powered by Claude Haiku, the coach detects 10 specific cognitive distortions (e.g., catastrophizing, all-or-nothing thinking).

Distortion Badges: Automatically labels detected patterns and surfaces tailored affirmations in indigo-glow bubbles.

Edge Intelligence: Backend deployed via Cloudflare Workers with Llama 3.3 for sub-second latency.

2. Deep Gamification (XP System)
Progressive Levels: 10 levels from Seeker → Ascendant.

XP Engine: Earn points for journaling, breathing, evidence logging, and chatting.

Live Dashboard: Home screen features a real-time XP progress bar and task chips that fill with accent color upon completion.

3. Biometric & Multi-Device Integration
HealthKit + Oura: Utilities for HRV and resting heart rate monitoring. detectStress() flags physiological markers to trigger generateResponsiveAffirmation().

Apple Watch (watchOS): Three-tab companion app for affirmations, haptic-guided box breathing, and streak tracking.

iOS Widgets: System-wide integration via Small and Medium formats for instant mindset resets.

4. Mindful Practices
Breathing Exercise: 4 patterns (Box, 4-7-8, Calm, Energize) with haptic feedback and Reanimated-driven visuals.

Evidence Journal: A "micro-evidence" database where users prove their affirmations true, supported by AI reflection.

Audio Motivation: On-demand motivational blurbs generated by Claude and read via expo-speech.

# 🛠️ Technical Stack
Frontend: React Native / Expo (SwiftUI for Watch/Widgets)

AI/LLM: Claude Haiku (CBT Logic) + Llama 3.3 (Edge Affirmations)

Infrastructure: Cloudflare Workers + Durable Objects (Stateful Memory Layer)

Persistence: CoreData (Native) + Shared App Groups for Watch/Widget synchronization.

Biometrics: Apple HealthKit + Oura API integration.

Animations: React Native Reanimated + Expo Video.

# 📦 Installation
Bash
# Clone the repository
git clone https://github.com/megangadiparthy/alta-affirmation-app.git

# Install dependencies
npm install

# Start the development server
npx expo start
Note: For the Watch app and HealthKit features, open the ios/ folder in Xcode 15.0+ and ensure the proper Capabilities are added.
